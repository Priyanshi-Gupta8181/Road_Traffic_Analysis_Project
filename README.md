# 🚦 Traffic Flow Analysis and Accident Prediction using Machine Learning

## 📌 Project Overview
This project was developed as a **Mini Project during B.Tech Second Year (Semester 4)** as part of the Computer Science and Engineering (Data Science) curriculum.

The objective of this project is to analyze road accident data, identify important factors contributing to accidents, and predict accident severity using Machine Learning algorithms.

By leveraging historical accident data, the system helps in understanding traffic patterns and supports decision-making for improving road safety.

---

## 🎯 Objectives
- Analyze historical road accident data.
- Identify major factors affecting accident severity.
- Perform data preprocessing and feature engineering.
- Train Machine Learning models for accident severity prediction.
- Compare the performance of different ML algorithms.
- Provide insights through visualizations and analysis.

## 📂 Dataset
The project uses a road accident dataset containing information such as:

- Weather Conditions
- Road Surface Conditions
- Speed Limit
- Number of Vehicles
- Number of Casualties
- Light Conditions
- Month and Year
- Accident Severity

The dataset was cleaned and preprocessed before model training.

---

## ⚙️ Machine Learning Models Used
The following machine learning algorithms were implemented and evaluated:

1. Logistic Regression
2. Decision Tree Classifier
3. Random Forest Classifier

After comparison, the best-performing model was selected for prediction.
